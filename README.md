# URL Shortener API

A simple RESTful API for URL shortening using Node.js, Express, and MongoDB.

## Features
- Shorten long URLs
- Retrieve original URLs
- Update and delete short URLs
- Track access statistics

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Nanoid
